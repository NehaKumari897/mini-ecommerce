# mini-ecommerce
A fully functional mini e-commerce web application built with React that demonstrates modern web development practices including component design, state management, API integration, and responsive UI design.



Key Features:

Product Listing: Displays 20 products in a responsive grid layout fetched from Fake Store API
Search Functionality: Real-time search to find products by name
Category Filter: Filter products by different categories
Price Sorting: Sort products from Low to High or High to Low
Shopping Cart: Full cart management with add, remove, and quantity update features
Stock Management: Shows stock status and prevents adding out-of-stock items
Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
Empty States: Proper handling of empty cart and no products found scenarios

Technologies Used:

React (Functional Components with Hooks)
Tailwind CSS (for styling)
Lucide React (for icons)
Fake Store API (for product data)
JavaScript ES6+
